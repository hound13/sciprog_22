File Hanoi.c
Prints on screen the sequence of moves required to solve the Tower of Hanoi for any given number n

Function Hanio_Step move n discs from rod 1 to rod 3, using rod 2 as an intermediary.  If n = 1, then it moves the disc from rod 1 to rod 3. Otherwise, it  moves n-1 discs from rod 1 to rod 2, using rod 3 as an intermediary, and then prints the sequence, and then moves n-1 discs from rod 2 to rod 3, using rod 1 as an intermediary
